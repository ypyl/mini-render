## Why

The demo site at `ypyl.github.io/mini-render` has no link back to the GitHub repository. Visitors who discover the demo have no easy path to view the source code, open issues, or star the repo. Adding a prominent repo link on the home page bridges the demo and the source.

## What Changes

- **HomePage** displays a link to `https://github.com/ypyl/mini-render` below the description, styled as a subtle text link with an icon.
